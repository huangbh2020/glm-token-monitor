# 果冻精灵 "Sparky" 宠物形象设计

## 概述

重构宠物 SVG 形象，采用 Duolingo 可爱扁平风格，创建一个简约的代码精灵吉祥物。

## 设计决策

| 属性 | 决策 |
|------|------|
| 名称 | Sparky（火花精灵） |
| 风格 | Duolingo 可爱扁平风 |
| 形象 | 小幽灵/精灵形态，果冻质感 |
| 颜色 | 5 状态动态变色（Fresh/Flow/Warning/Panic/Dead） |
| 动画 | 5 种状态动画 |
| 表情 | 简单圆点眼 |

## 基础结构

SVG 元素数量：4-5 个基础图形

```
身体（主体）     - 椭圆/水滴形，圆点眼在身体内部
眼睛左           - 黑色圆点
眼睛右           - 黑色圆点
高光（可选）     - 身体顶部小椭圆，增加果冻质感
```

## 尺寸与比例

- viewBox: `0 0 64 64`
- 身体：椭圆 `cx="32" cy="36" rx="24" ry="26"`
- 眼睛：两个圆 `r="3"`，间距 12px，位于身体中上部

## 颜色系统

| 状态 | 身体填充 | 身体描边 | 眼睛 |
|------|----------|----------|------|
| Fresh | `#10B981` | `#059669` | `#1F2937` |
| Flow | `#3B82F6` | `#2563EB` | `#1F2937` |
| Warning | `#F59E0B` | `#D97706` | `#1F2937` |
| Panic | `#EF4444` | `#DC2626` | `#1F2937` |
| Dead | `#9CA3AF` | `#6B7280` | `#6B7280` |

## 动画规格

| 状态 | 动画效果 | SMIL 实现 |
|------|----------|-----------|
| Fresh | 平静呼吸 + 眼睛慢眨 | scale + ellipse ry |
| Flow | 轻微摇摆 + 正常眨眼 | rotate + ellipse ry |
| Warning | 紧张抖动 + 眼睛放大 | translate + circle r |
| Panic | 快速弹跳 + 眼睛瞪大 | translate + circle r |
| Dead | 无动画，颜色变灰 | - |

## 表情变化

通过眼睛大小和间距表达情绪：

- **开心/Fresh**：眼睛正常圆点，略微靠拢
- **Flow**：眼睛正常，间距标准
- **紧张/Warning**：眼睛略大，间距正常
- **恐慌/Panic**：眼睛最大，可加小星星特效
- **困倦/Dead**：眼睛变小或变成横线
