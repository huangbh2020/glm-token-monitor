# PlanGuard 系统图标设计方案

## 设计理念

基于你的需求（专业简洁、独立品牌、监控仪表盘核心），结合项目现有的颜色系统，设计了以下四个方案。

## 方案对比

### 方案一：环形进度仪表盘
**文件**: `方案1-环形进度.svg`

**特点**:
- 最直观的"用量监控"表达
- 使用项目的绿色→红色渐变系统
- 中心显示百分比，信息清晰
- 适合作为系统托盘图标

**适用场景**:
- Windows 悬浮窗
- 系统托盘
- 应用程序图标

**优势**:
- 简洁专业，符合工具类应用定位
- 颜色系统与项目状态一致
- 可读性强，小尺寸下依然清晰

---

### 方案二：饼图仪表盘
**文件**: `方案2-饼图仪表.svg`

**特点**:
- 分段显示不同状态颜色（绿/蓝/黄/橙/灰）
- 动态指示器显示当前状态
- 包含状态标签（FLOW）

**适用场景**:
- 设置面板
- 状态详情页
- 大尺寸展示

**优势**:
- 信息密度高，一眼看出所有状态
- 视觉丰富但不失专业感
- 适合作为应用内图标

---

### 方案三：仪表指针风格
**文件**: `方案3-仪表指针.svg`

**特点**:
- 深色背景，科技感强
- 经典仪表盘设计，指针指向当前用量
- 包含品牌标识 "PLANGUARD"

**适用场景**:
- 桌面应用主图标
- macOS 风格的应用图标
- 需要强调品牌识别的场景

**优势**:
- 最经典的"监控"隐喻
- 深色设计在 macOS 上很协调
- 品牌识别度最高

---

### 方案四：P字母+进度环组合
**文件**: `方案4-P字母进度.svg`

**特点**:
- 字母 P（PlanGuard）与进度环结合
- 右上角百分比徽章
- 底部进度条指示器

**适用场景**:
- 应用程序主图标
- 启动器图标
- 品牌推广场景

**优势**:
- 品牌识别度高
- 设计感强，现代简洁
- 多元素组合，层次丰富

## 推荐方案

### 主图标推荐：**方案一或方案四**

**理由**:
- 方案一：最符合"监控工具"定位，简洁专业
- 方案四：品牌识别度最高，适合作为应用主图标

### 系统托盘推荐：**方案一**

**理由**:
- 小尺寸下可读性强
- 圆形设计适合托盘区域
- 颜色状态一目了然

### macOS 推荐：**方案三**

**理由**:
- 深色设计与 macOS 系统风格一致
- 仪表盘设计经典专业
- 包含品牌标识，识别度高

## 使用建议

### 1. 多尺寸导出
建议导出以下尺寸：
- **Windows**: 16x16, 32x32, 48x48, 256x256
- **macOS**: 16x16, 32x32, 128x128, 256x256, 512x512, 1024x1024
- **Web**: 192x192 (PWA), 512x512

### 2. 状态变体
可以根据当前使用状态生成不同颜色的图标变体：
- **Fresh (0-30%)**: 绿色主调
- **Flow (31-60%)**: 蓝色主调
- **Warning (61-80%)**: 黄色主调
- **Panic (81-95%)**: 橙色主调
- **Dead (96-100%)**: 灰色主调

### 3. 动态效果
对于 Windows 悬浮窗，可以添加：
- 进度环动画
- 状态变化时的过渡动画
- 脉冲效果（Panic 状态）

### 4. 文件格式
- **SVG**: 用于开发和高分辨率场景
- **PNG**: 用于系统图标
- **ICO**: Windows 图标
- **ICNS**: macOS 图标

## 技术实现

### Tauri 配置

更新 `src-tauri/tauri.conf.json`:

```json
{
  "bundle": {
    "icon": [
      "icons/32x32.png",
      "icons/128x128.png",
      "icons/128x128@2x.png",
      "icons/icon.icns",
      "icons/icon.ico"
    ]
  }
}
```

### 动态图标（可选）

在 Rust 端实现状态变化时更新托盘图标：

```rust
// src-tauri/src/tray.rs
use tauri::Manager;

pub fn update_tray_icon(app: &tauri::AppHandle, state: &str) {
    let icon_path = match state {
        "Fresh" => "icons/icon-green.png",
        "Flow" => "icons/icon-blue.png",
        "Warning" => "icons/icon-yellow.png",
        "Panic" => "icons/icon-orange.png",
        "Dead" => "icons/icon-gray.png",
        _ => "icons/icon.png",
    };
    // 更新托盘图标
}
```
